# C Practice Problems

A comprehensive collection of C programming practice problems, organized by days, covering everything from basic syntax to complex logic and mathematical operations.

## 🚀 Overview

This repository contains structured practice problems designed to strengthen C programming skills. It includes daily challenges, a PDF guide for fast-track learning, and modular code implementations.

## 📁 Repository Structure

The project is organized into daily folders, each typically containing:
- `main.c`: The entry point with menu-driven testing interfaces.
- `Operation.c`: Implementation of the logic for the specific problems.
- `DayN.h`: Header files containing function prototypes and constants.

```text
.
├── Day1 Practice Problems/      # Basics: conversions, shifts, and simple logic
├── Day2 Practice Problems/      # Loops: Palindromes, Primes, Armstrong numbers
├── Day3_4 Practice Problems/    # Intermediate logic and algorithms
├── Day5 Practice Problems/      # Advanced practice
├── Day6 Practice Problems/      # Continued practice
├── Day7 Practice Problems/      # Complex implementations
├── Fast Track C Practice problems.pdf  # Reference guide
└── README.md
```

## 🛠️ Key Features

### Day 1: The Basics
- Character case conversion (`to_lowercase`)
- Geometry calculations (Circle Area)
- Temperature conversion (Celsius to Fahrenheit)
- Bitwise operations (Left shift)
- Basic parity and leap year checks

### Day 2: Numerical Logic & Patterns
- **Number Theory**: Prime numbers, Armstrong numbers, Amicable pairs, and Palindromes.
- **Conversions**: Binary to Decimal and Decimal to Binary.
- **Volume Calculations**: Cube, Cuboid, Sphere, Cylinder, and Cone.
- **Utilities**: Electricity bill calculator and sequence generators.
- **Patterns**: Half pyramid and inverse half pyramid implementations.

### Days 3-7
Detailed implementations of intermediate to advanced C concepts including further mathematical sequences and data manipulation.

## 💻 How to Run

To compile and run any of the daily problems, navigate to the specific directory and use a C compiler like `gcc`:

```bash
cd "Day1 Practice Problems"
gcc main.c Operation.c -o day1
./day1
```
